# MMKApplication

MMKApplication is a .net core application used for message management.

## Installation
```
# install .net6.0 framework to run locally

# Restore every dependencies including 
- stackExchangeRedis v2.5.43
- npqsl v6.0.3
- Dapper ORM v2.0.123
```
